LeafQuadtree

class LeafQuadtree[T] extends SpatialTree[T]

An immutable LeafQuadtree implementation

A quadtree is a trie with 4 children nodes per parent used to partition a cartesian plane and speed up object-object interactions in graphical environments.

T

The type of objects contained by the quadtree. They must be PhysicalObject

See also

http://gamedevelopment.tutsplus.com/tutorials/quick-tip-use-quadtrees-to-detect-likely-collisions-in-2d-space--gamedev-374
