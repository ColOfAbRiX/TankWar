  1. trait BehaviourBuilder extends AnyRef

    Neural Network Behaviour Builder

    Neural Network Behaviour Builder

    Neural networks are considered made of 3 parts:

    • A behaviour: feed-forward, Elman, Jordan, ...
    • A structure: 3-layers, perceptron, random, ...
    • Data: biases, weights, activation functions, ...

    A behaviour is used directly by the client to construct its own NN and which can know only the number of inputs it provides and the number of output it needs

  2. trait DataReader extends AnyRef

    Data Reader

    Data Reader

    It provides a structured access to the data of the Neural Network. It works in a sequential fashion so that every call of nextLayerReader returns the current LayerReader and moves forward an internal pointer to the next one.
