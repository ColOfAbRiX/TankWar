  2. class SpatialHash[T] extends SpatialSet[T]

    An object to index shapes in space.

    An object to index shapes in space. It implements the concept of Spatial Hashing, Spatial hashing is a process by which a 3D or 2D domain space is projected into a 1D hash table. The data structure has been implemented to be as efficient as possible and from my measurements it can be twice as fast as the plain brute force collision detection.

    T

    The type of object that this collection will contain. Must have a conversion to SpatialIndexable
