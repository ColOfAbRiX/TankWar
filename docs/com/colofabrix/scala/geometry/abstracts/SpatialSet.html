SpatialSet

Related Doc: package abstracts

abstract class SpatialSet[T] extends AnyRef

A SpatialSet is a set of objects that can be located in a 2D Space.

This trait is used to implement efficient ways for storing and retrieving objects in the cartesian plane, in particular for collision detection. Some of its implementation implementation are the com.colofabrix.scala.geometry.collections.SpatialTree, the com.colofabrix.scala.geometry.collections.SpatialHash and also the fake com.colofabrix.scala.geometry.collections.DummySet for testing.

T

The type of objects contained in the Set
