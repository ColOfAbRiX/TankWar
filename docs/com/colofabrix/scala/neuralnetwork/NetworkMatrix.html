NetworkMatrix

class NetworkMatrix extends Matrix[Double]

An adjacency matrix for Neural Networks

It is a weighted, asymmetric adjacency matrix with Double.NaN to indicate non-connected vertices. The matrix is composed of two parts, a square matrix that contains the weights of the edges and the last row that contains the biases for each neuron
