abstract class InputHelper[T] extends AnyRef

Input helper object

Is is used to provide a nice interface between the input of a GenericNeuralNetwork (s Seq[T]) and the client. Using a companion object with arguments will bring much cleaner code can create a tidy way to provide data to the GenericNeuralNetwork
