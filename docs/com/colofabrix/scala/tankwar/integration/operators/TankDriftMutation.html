  18. def mutationRule(scale: Double)(x: Double, rng: Random): Double

    Rules that defines how to mutate

    Rules that defines how to mutate

    This rule works like this: for every value there is a specific probability that it mutates. If the value mutates, then the new value a drift from the old value and there are no random possibilities. Usually the new value should have higher probability to fall near the old value with a Gaussian distribution, but this is up to the developer

    scale

    Range of the random values, in [-scale, scale]

    x

    Value that possibly mutates

    rng

    Random number generator

    returns

    The old value or a new mutated value
